SBML file structure details

The SBML file must contain information about gene associations and metabolic pathways. They must be specified in the Notes tag of the reactions in the SBML file as follows:

  • For gene associations, TIM-Viz recognizes the string "p> GENE_ASSOCIATION:". Then genes must be separated by the "or" keyword.
    Example of a correct way of specifying gene associations: 
    <reaction id="R1">
    <notes>
        <p>GENE_ASSOCIATION: (gene1 or gene2 or gene3)</p>
    </notes>
...
  • For metabolic pathways, TIM-Viz recognizes the string "p> SUBSYSTEM:". If a reaction belongs to multiple metabolic pathways, they must be separated by ";".
    Example of a correct way of specifying pathways:
    • <reaction id="R1">
    <notes>
        <p>SUBSYSTEM: pathway1 ; pathway2</p>
    </notes>
...
It is recommanded that a maximum of reactions have gene associations and metabolic pathways associated.
